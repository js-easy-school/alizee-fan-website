# Alizée Fan Website

This is a gift website created for French singer Alizée.

## Files needed for hosting

**Main file:** `index.html` - This is the complete website in a single file.

## How to deploy

1. Upload `index.html` to your web hosting
2. Rename it to `index.html` if needed
3. Done! The website is ready

## What NOT to upload

- `.github/` folder - This is only for GitHub Actions, not needed for hosting
